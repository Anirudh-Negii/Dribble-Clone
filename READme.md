# Dribbble Clone

🔗 **Live Demo:** https://dribble-clone-84.vercel.app

This project is a recreation of the Dribbble landing page built using HTML and CSS.


## Goal

The objective of this assignment was to improve:

* HTML page structuring
* CSS layout building
* Flexbox concepts
* Positioning and layering
* CSS animations and transitions
* UI recreation skills
* Attention to spacing and visual hierarchy
* Research and frontend problem-solving skills

## Tech Used

* HTML5
* CSS3
* Remix Icons

## Usage

Open the `index.html` file in your browser.
